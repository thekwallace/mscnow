MSCNow

Quick MSC cue generation for macOS. Create and send MIDI Show Control GO cues in seconds.

Download for macOS

Features

Wizard-Style UI

Step through cuelist selection, cue entry, and confirmation in a clean, focused interface.

Smart Cue Entry

Enter single cues, comma-separated lists, or sequential ranges like 1-10.

Live MIDI Output

Send cues directly to any MIDI device with configurable delays between commands.

Export to MIDI

Generate standard MIDI files with Pro Tools compatibility for offline editing.

Quick Open

Create temporary MIDI files and open them directly in your DAW with one click.

Flexible Configuration

Configure device ID and command format for lighting, sound, machinery, and more.

How It Works

1

Enter Cuelist

Specify which cuelist the cues belong to.

2

Enter Cues

Type cue numbers as a single cue (5), list (1, 5, 10), or range (1-20).

3

Send or Export

Send live via MIDI, open in your DAW, or export as a MIDI file.
